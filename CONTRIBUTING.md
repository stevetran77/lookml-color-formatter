# Contributing to LookML Color Formatter

Thank you for your interest in contributing! Here's how you can help.

## Development Setup

1. Clone the repository
```bash
git clone https://github.com/stevetran77/lookml-color-formatter.git
cd lookml-color-formatter
```

2. Install dependencies
```bash
npm install
```

3. Open in VS Code
```bash
code .
```

4. Run in debug mode
- Press `F5` to launch the Extension Development Host
- The extension will run in a new VS Code window

## Making Changes

1. Edit files in `src/` directory
2. TypeScript will compile automatically (or run `npm run watch`)
3. Reload the extension in the debug window (`Ctrl+R`)
4. Test your changes

## Testing

- Open a `.lkml` or `.lookml` file in the debug window
- Verify syntax highlighting works correctly
- Test formatting with `Shift+Alt+F` (Format Document)

## Submitting Changes

1. Create a new branch
```bash
git checkout -b feature/your-feature-name
```

2. Commit your changes
```bash
git commit -m "Add description of changes"
```

3. Push to your fork and submit a pull request

## Code Style

- Use 2-space indentation
- Follow existing code patterns
- Keep functions focused and simple

## Bug Reports

Found a bug? Open an issue on [GitHub](https://github.com/stevetran77/lookml-color-formatter/issues) with:
- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)

## Questions?

Feel free to open an issue or discussion on GitHub.
